Getting started

First launch

When you first launch AgentPrime, the onboarding wizard walks you through initial setup.

Opening a project

  1. Press Ctrl+O (Windows/Linux) or Cmd+O (macOS).
  2. Or open the Command Palette (Ctrl+K) and search for “Open Project”.
  3. Select a folder to use as your workspace.

Setting up AI providers

  1. Open Settings (Ctrl+, or Command Palette > Settings).
  2. Go to AI Providers.
  3. Add API keys for the providers you use:
    • Anthropic — console.anthropic.com
    • OpenAI — platform.openai.com
    • Ollama — runs locally; no key required
    • OpenRouter — openrouter.ai

AI features

AI Composer

Your main surface for AI: answer questions, generate or refactor code, debug, and explain logic.

Ghost text completions

Inline suggestions appear as you type. Press Tab to accept.

Mirror Intelligence

Learns from your patterns and suggests completions aligned with your style.

Words to Code

Describe what you want in natural language; AgentPrime scaffolds the structure for you.

Project management

Task Manager

Use Ctrl+Shift+T to create tasks, set priorities, and mark work complete.

Git integration

The Git panel (Ctrl+Shift+G) supports staging, commits, branches, and push/pull.

Settings

Theme

Switch between light and dark themes in Settings.

Font size

Editor font size is adjustable (typically 10–24pt).

AI model selection

  • Fast model — quick tasks and completions
  • Deep model — deeper analysis and refactors

Troubleshooting

AI not responding

  1. Confirm your API key in Settings.
  2. Check your network connection.
  3. Review the provider’s status page.

Performance issues

  1. Close unused files and panels.
  2. Restart AgentPrime.
  3. Check RAM and CPU usage on your machine.

Files not saving

  1. Verify file permissions.
  2. Ensure you have free disk space.
  3. Try Save As to another location.

Getting help

  • Use the Command Palette (Ctrl+K) to find commands.
