Start here

When no file is open, AgentPrime shows the welcome screen. The default Classic View is the clean centered launch surface. You can switch to Launchpad View if you want the larger product-style home screen.

Classic View

Best for fast daily use: New Project, Open Project, AI Chat, New File, recent projects, and core shortcuts.

Launchpad View

Best for demos or onboarding: larger hero, Target Loop, proof cards, and the same creation actions.

The views are optional.

Switching views does not change project creation, Agent Mode, or workspace behavior. It only changes the welcome screen layout.

Create projects

Use New Project from the welcome screen or press Ctrl+Shift+N. AgentPrime opens the template gallery, collects project details, writes files, installs dependencies when supported, then opens the generated workspace.

Choose a templatePick desktop, full-stack, frontend, backend, or game starters. Complex briefs can route into AI Composer.
Configure the projectSet the project name, destination, author, and description. AgentPrime sanitizes project folder names safely.
Scaffold filesThe template engine writes the starter files and rolls back cleanly if materialization fails.
Install dependenciesPost-create commands run when supported. Install warnings are shown as setup notes, not as failed creation.
Open and launchThe created workspace opens automatically. The success screen can launch a preview when the project runner detects a URL.
If dependencies need attention:

The files are still created. Read the setup note, open the workspace, and run the suggested install command in the terminal.

Agent Mode

Agent Mode is the file-aware build lane. Open it with Ctrl+L or the Ask AI button. It can inspect context, propose changes, run checks, and hand changes to review before applying.

Agent Mode

Use for edits, refactors, repairs, project generation, and multi-step implementation.

Just Chat

Use for brainstorming, explanations, and non-editing questions.

Dino Buddy

Use for calmer guidance, playful support, and learning-style help.

Review Loop

Review generated diffs, accept or reject files, verify, run, repair, and revert when needed.

Workspace flow

Open a project

Use Ctrl+O or the welcome screen Open Project action. The file explorer, Git panel, editor, terminal, status bar, and Agent Mode all use the active workspace path.

Run and preview

Press F5 or use project run actions. When AgentPrime gets a local URL from the project runner, Live Preview opens in the IDE.

Status bar

The bottom bar summarizes AI connection, current model, desktop/Python Brain status, doctor readiness, file language, line count, shortcuts, and time.

Settings

Open Settings from the top bar or Command Palette. The main areas are editor preferences, appearance, AI provider credentials, Agent Mode behavior, shortcuts, and advanced diagnostics.

AI providers

Configure Ollama, OpenAI, Anthropic, or OpenRouter. AgentPrime can show provider status and credential setup notes.

Agent behavior

Choose behavior profiles, autonomy level, specialized agents, and model routing preferences.

Editor

Control font size, tab size, word wrap, minimap, line numbers, auto-save, and inline completions.

Diagnostics

Use advanced startup diagnostics when tools, provider setup, or runtime health needs attention.

Troubleshooting

Project created but dependencies did not install

This usually means a tool is missing, the package manager failed, or the network blocked install. The project files are still available. Open the workspace and run the install command from the terminal.

Preview did not launch

Some project types need a manual command first, such as npm run dev. Check the template README or terminal output.

AI is not responding

  1. Open Settings and confirm provider credentials.
  2. Check the status bar for provider/model state.
  3. Open System Status for startup or doctor warnings.

Workspace looks stale

Reopen the folder, use the explorer refresh path, or restart AgentPrime if the file watcher fell behind.

Getting help

Use the Command Palette with Ctrl+K to discover commands. For technical details, see the project GitHub repository.

Best first prompt for Agent Mode:
